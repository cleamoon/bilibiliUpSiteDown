# bilibiliUpSiteDown
Downloader to download all videos of a video uploader on `Bilibili.com`.

## Requirement
* Python 
* Selenium
* Chromedriver
* You-get

## Usage: 

```shell
source venv/bin/activate
pip3 install -r requirements.txt
python3 ./bilibiliUpSiteDown <numberIdOfaUp>
```
