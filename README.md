# bilibiliUpSiteDown
Downloader to download all videos of a video uploader on `Bilibili.com`.

## Requirement
* Python 
* Selenium
* Chromedriver
* You-get

## Usage: 
bilibiliUpSiteDown <numberIdOfaUp>
